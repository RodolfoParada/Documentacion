**Instalar vue clic**
```vue
sudo npm install -g @vue/cli
```
**Crear Proyecto vue clic**
```vue
vue create nombre-del-proyecto
```
**Instalar VUEX**
```vue
npm install vuex@next
```
**Instalar Bootstrap**
```vue
npm install bootstrap
```
**Instalar vue vite**
```vue
 sudo npm init vite@latest
```
