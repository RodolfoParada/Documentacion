### Instalar angular 
**Para crear un proyecto en Angular**
```
npm install -g @angular/cli
```

### Se debe crear un una carpeta antes para crear un proyecto en angula 
**Para crear un proyecto en Angular**
```
ng new nombre-de-proyecto
```
**Para levantar el proyecto angular en el navegador**
```
ng serve -o
```
